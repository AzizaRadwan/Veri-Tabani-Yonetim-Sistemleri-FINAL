# MailSense AI Frontend Dashboard

This folder contains the web interface of the MailSense AI project.

The dashboard is designed to present the results of AI-powered email sentiment, risk, and trend analysis in a user-friendly way.

## Files

| File | Description |
|---|---|
| index.html | Main dashboard page |
| style.css | Dashboard styling file |
| app.js | Static demo data and dynamic table rendering |

## Dashboard Features

The dashboard displays:

- Total analyzed emails
- Negative email count
- High-risk email count
- Average risk score
- Sentiment distribution
- High and critical risk emails
- Most frequent keywords
- System alerts
- Workflow logs

## Usage

Open the `index.html` file in a web browser.

No installation is required for the static demo version.

## Future Improvements

In future versions, this interface can be connected to a backend API or directly integrated with n8n and SQL Server data outputs.
